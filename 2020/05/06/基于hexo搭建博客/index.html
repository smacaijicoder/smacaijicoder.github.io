hexo基本配置

在文件根目录下的_config.yml，就是整个hexo框架的配置文件了。

网站
参数 描述
title 网站标题
subtitle 网站副标题
description 网站描述，建议包含网站的关键字
author 作者
language 语言
timezone 网络时区
网址
参数 描述
url 网址
root 网站根目录
permalink 文章的永久链接格式
permalink_defaults 永久链接各部分默认值

Front-matter

Front-matter 是文件最上方以 --- 分隔的区域，用于指定个别文件的变量，举例来说：

下是预先定义的参数，您可在模板中使用这些参数值并加以利用。

参数 描述
layout 布局
title 标题
data 建立日期
updated 更新日期
comments 开启文章评论功能
tags 标签
categories 分类
permalink 覆盖文章网址

其中，分类和标签需要区别一下，分类具有顺序性和层次性，也就是说 Foo, Bar 不等于 Bar, Foo；而标签没有顺序和层次。
