icarus verilog使用简介

一款轻量版的verilog仿真平台，很适合初学者及小型化仿真需求者：Icarus Verilog + gtkwave。前者可以认为是个仿真器，后者是个看波形的软件。

Icarus Verilog开源，可支持windows和linux。官网下载地址： http://bleyer.org/icarus/

使用教程

  • 编写 Verilog代码和testbench。注意：testbench中需要以下两句，这两行主要是给gtkwave这个工具使用的

  • 编译仿真

    1. 打开命令行，cd 进入 .v文件所在的文件夹（或直接在该文件夹的地址栏键入cmd）

    2. 执行 iverilog -o dsn icarus_example.v得到文件dsn. 要将要编译的.v文件全部写入

    3. 执行 vvp dsn（此处的dsn与（2）一致）

  • 用gtkwave打开vcd仿真波形

    执行 gtkwave test.vcd (vcd的名字是在testbench中命名的，此处为 test.vcd) 打开仿真波形

Ubuntu 环境

  1. 下载并安装软件，需要联网。在root权限下，
    执行 apt-get install Verilog
    执行 apt-get install gtkwave
  2. 编写 Verilog代码和testbench。
  3. 编译仿真 (同window环境)
    a) 执行 iverilog -o dsn icarus_example.v得到文件dsn.
    b) 执行 vvp dsn
  4. 用gtkwave打开vcd仿真波形
    a) 执行 gtkwave test.vcd (vcd的名字是在testbench中命名的，此处为 test.vcd) 打开仿真波形
